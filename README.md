# mysteryadventurebot

Game "The Mystery Adventure Bot" — permainan petualangan teks horror bertema rumah hantu terbengkalai di daerah Yogyakarta.

Menjalankan:

1. Pastikan Python 3.8+ terpasang.
2. Jalankan:

```bash
python main.py
```

Kontrol:
- `explore` untuk menjelajahi lokasi saat ini.
- `move` untuk pindah ke lokasi lain.
- `inventory` untuk melihat barang.
- `quests` untuk melihat quest yang harus diselesaikan.
- `quit` untuk keluar.

Selamat bermain — jangan sampai kewarasanmu habis!
